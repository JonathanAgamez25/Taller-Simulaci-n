# 🎲 Simulación Monte Carlo - Juego Yahtzee

## 📌 Descripción

Este proyecto implementa una simulación del juego **Yahtzee** utilizando el método de **Monte Carlo**, con el objetivo de analizar probabilidades y resultados posibles a través de múltiples ejecuciones.

Fue desarrollado como parte de un taller académico en el área de simulación.

---

## 🧠 ¿Qué es Monte Carlo?

El método de Monte Carlo es una técnica que utiliza **simulaciones aleatorias** para resolver problemas que pueden ser determinísticos en principio.

En este proyecto se usa para:

* Simular lanzamientos de dados 🎲
* Analizar combinaciones posibles
* Evaluar resultados del juego

---
